jsloan
============

### 대출이자계산 API(JAVA)
jsloan은 Non-repository(DB) POJO로 이루어진 대출이자계산 API 입니다.
대출 정보를 참조해서 이자를 계산합니다. 

### 관련 기능
- 대출상환계획 (Plan) Scheduling
- 입금내역에 따른 상환내역 산출
- 연체판단 및 연체이자 계산
- DI:여러 조건에 따른 이자계산방법 주입토록 구성

### 사용된 기술정보
* JAVA - JDK 1.8
* Dependency, Build - Maven
* 기타 - lombok, JUnit4(for test)

### 사용한 IDE 
- Eclipse
